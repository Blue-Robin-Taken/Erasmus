# Setup

## Running

```
python3.6 -m venv venv
source venv/bin/activate
pip install --process-dependency-links -r requirements/run.txt
```

## Development

```
python3.6 -m venv venv
source venv/bin/activate
pip install -U --process-depenency-links -r requirements/dev.txt
```
